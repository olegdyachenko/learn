инициализиурем каталог 

git init

Затем в него клониурем проект

git add . - добавить изменения в папке в систему гита

git status - посмотреть изменения или статус текущего проекта

git commit -m "text" - это мы фиксиурем коммит

git push - Отправляем на сервер

git pull - скачиваем к себе изменения
